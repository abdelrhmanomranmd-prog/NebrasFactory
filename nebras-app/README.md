# تطبيق نبراس للجوال

تطبيق **عملاء · مناديب · مدير مبيعات** مبني على نفس المنصة الحية — أي تعديل في الإدارة يظهر فوراً في التطبيق.

## طرق التشغيل

### 1) من المتصفح (فوري)
افتحي: **https://www.nebrasplasticcompany.com/?app=1**

### 2) تثبيت PWA
من Chrome/Safari: **إضافة إلى الشاشة الرئيسية** — يستخدم `site.webmanifest` مع `start_url=/?app=1`

### 3) تطبيق المتجر (Capacitor)

```bash
cd nebras-app
npm install
npx cap add android   # مرة واحدة
npx cap add ios       # مرة واحدة (على Mac)
npx cap sync
npx cap open android
```

التطبيق يحمّل المنصة من السيرفر (`server.url`) — **لا حاجة لإعادة نشر التطبيق عند تعديل المنتجات أو العروض**.

## شريط التنقل حسب الدور

| الدور | التبويبات |
|--------|-----------|
| زائر / عميل | رئيسية · متجر · معرض · حسابي · عرض سعر |
| عميل مسجّل | رئيسية · متجر · معرض · لوحتي · عروضي |
| مندوب مبيعات | رئيسية · متجر · عروضي · عملاء · لوحتي |
| مدير مبيعات | رئيسية · متجر · العروض · الفريق · لوحتي |

## الملفات في المشروع الرئيسي

- `js/nebras-mobile-app.js` — منطق التطبيق وشريط التبويب
- `css/54-nebras-mobile-app.css` — تنسيق شريط التطبيق
- `site.webmanifest` — PWA
