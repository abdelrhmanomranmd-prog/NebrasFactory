■ الأخطاء التي كسرت الموقع (بعد تعديل الداشبورد)
───────────────────────────────────────────────────────────────
1) خطأ JavaScript — فاصلة ناقصة في DASHBOARD_HANDLER_MAP
   → الملف كله لا يعمل (فروع + صور + كل شيء)

2) سطر مكرر function openCustomSectionItem
   → نفس المشكلة

3) Supabase/localStorage فيه branches = []
   → الفروع تختفي بعد التحميل

4) مسارات صور CSS كانت images/ بدل /images/
   → 404 لخلفيات البطاقات

5) displayBranches كان يستخدم امتداداً واحداً فقط (.jpg)
   → صور فروع .webp لا تظهر

■ ما تم إصلاحه الآن (على Desktop)
───────────────────────────────────────────────────────────────
✅ بناء JavaScript سليم (node --check)
✅ ensureBuiltinBranches — 7 فروع دائماً
✅ displayBranches — يجرّب jpg/jpeg/png/webp/avif تلقائياً
✅ buildUrlList + imageUrlsFromSource — مسارات /images/ صحيحة
✅ حماية من branches فارغ في Supabase
✅ شركاؤنا + اعتمادات — ما زالت موجودة (التعديل الأول)
✅ إغلاق overlay يعيد ضبط المعرض (لا يكسر الأيقونات الأخرى)
