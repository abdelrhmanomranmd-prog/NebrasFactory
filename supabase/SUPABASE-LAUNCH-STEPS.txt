═══════════════════════════════════════════════════════════════

المشروع على Supabase:
  URL: https://oedldllrjavofpeaputz.supabase.co

ملفات SQL بالترتيب (مجلد supabase):
  000-wipe-public-fresh-start.sql  ← اختياري (بداية نظيفة)
  001-nebras-platform-v2.sql       ← إلزامي
  002-storage-nebras-media.sql      ← إلزامي (رفع الصور)
  003-verify-setup.sql              ← تحقق فقط
  004-grant-anon-authenticated.sql  ← إلزامي إن ظهر 401 في Console

───────────────────────────────────────────────────────────────
الخطوة 1 — الدخول وإنشاء/فتح المشروع
───────────────────────────────────────────────────────────────
1. افتحي https://supabase.com/dashboard
2. سجّلي الدخول
3. افتحي مشروع nebras-factory (أو أنشئيه)
4. ارسلي لقطة شاشة للوحة المشروع (Project Overview)

───────────────────────────────────────────────────────────────
الخطوة 2 — مسح قديم (اختياري — أول مرة أو إعادة ضبط)
───────────────────────────────────────────────────────────────
1. من القائمة: SQL Editor
2. New query
3. انسخي محتوى: 000-wipe-public-fresh-start.sql
4. Run
5. يجب أن تظهر Success
6. ارسلي لقطة شاشة Success

───────────────────────────────────────────────────────────────
الخطوة 3 — إنشاء جداول المنصة
───────────────────────────────────────────────────────────────
1. SQL Editor → New query
2. انسخي محتوى: 001-nebras-platform-v2.sql
3. Run
4. Success
5. ارسلي لقطة شاشة

───────────────────────────────────────────────────────────────
الخطوة 4 — تخزين الصور (bucket)
───────────────────────────────────────────────────────────────
1. SQL Editor → New query
2. انسخي محتوى: 002-storage-nebras-media.sql
3. Run
4. Success
5. من القائمة: Storage → يجب أن ترى nebras-media
6. ارسلي لقطة Storage

───────────────────────────────────────────────────────────────
الخطوة 5 — التحقق
───────────────────────────────────────────────────────────────
1. SQL Editor → New query
2. انسخي: 003-verify-setup.sql
3. Run
4. النتيجة: 14 مفتاح في nebras_data_store + bucket nebras-media
5. ارسلي لقطة النتائج

───────────────────────────────────────────────────────────────
الخطوة 6 — مفاتيح API (تأكيد الربط)
───────────────────────────────────────────────────────────────
1. Settings → API
2. Project URL يجب أن يطابق الملف js/nebras-platform.js
3. anon / publishable key يطابق الملف
4. ارسلي لقطة (يمكن إخفاء جزء من المفتاح)

───────────────────────────────────────────────────────────────
الخطوة 7 — رفع ملفات الموقع (ليست داخل SQL)
───────────────────────────────────────────────────────────────
ملفات الموقع (index.html, css, js, images) تُرفع إلى:
  GitHub → Vercel  (أو أي استضافة ثابتة)

Supabase يخزّن:
  • بيانات المنصة (منتجات، أيقونات، إعدادات)
  • الصور التي ترفعينها من الإدارة

───────────────────────────────────────────────────────────────
الخطوة 8 — أول تشغيل حقيقي
───────────────────────────────────────────────────────────────
1. افتحي الموقع
2. الإدارة → NEBRASFACTORY / NEBRASFACTORYCOMPANYBASIC
3. إدارة محتوى الموقع → عدّلي منتجاً واحداً واحفظي
4. Supabase → Table Editor → nebras_data_store → site_products
   يجب أن يظهر JSON فيه المنتجات
5. ارسلي لقطة
